# L2: 機能仕様 - ステータスページ

## ページ構成

```
┌─────────────────────────────────────┐
│  ヘッダー（シンプル）                │
├─────────────────────────────────────┤
│  ページタイトル                      │
├─────────────────────────────────────┤
│  サービスカード                      │
│  ┌─────────────────────────────┐    │
│  │ サービス名 + ステータス       │    │
│  │ 概要                         │    │
│  │ CPU使用率グラフ              │    │
│  │ リンクボタン                 │    │
│  └─────────────────────────────┘    │
├─────────────────────────────────────┤
│  フッター                           │
└─────────────────────────────────────┘
```

---

## 1. ヘッダー

### 表示要素
- ロゴ（ホームへのリンク）
- シンプルなデザイン（ホームページとは異なる白ベース）

---

## 2. サービスカード

### 表示要素（1サービスあたり）
- サービス名
- ステータスインジケーター（稼働中: 緑、停止中: 赤）
- サービス概要（1〜2文）
- CPU使用率グラフ（折れ線グラフ）
- サービスへのリンクボタン

### CPU使用率グラフ
- 横軸: 時間（過去のデータポイント）
- 縦軸: CPU使用率（0〜100%）
- 1分ごとにデータを取得・蓄積
- ブラウザ側でデータを保持（サーバー側での履歴保存なし）
- ページを開いている間のみデータが蓄積される

### ステータス表示
| ステータス | 表示 | 色 |
|-----------|------|-----|
| 稼働中 | ● Operational | 緑 |
| 停止中 | ● Down | 赤 |

## 4. デザイン仕様

### カラーパレット（ステータスページ専用）
| 用途 | 色 | 備考 |
|------|-----|------|
| 背景 | 白（#ffffff） | ベース |
| テキスト | 黒（#1a1a1a） | 本文 |
| アクセント | ワインレッド（#722F37） | ボタン、リンク |
| サブアクセント | 黒（#0a0a0a） | 見出し |
| 稼働中 | 緑（#22c55e） | ステータス |
| 停止中 | 赤（#ef4444） | ステータス |
| カード背景 | ライトグレー（#f5f5f5） | カード |

### タイポグラフィ
- 見出し: 黒、太字
- 本文: ダークグレー
- 読みやすさ重視

---

## 5. 振る舞い

### 初期表示
1. ページ読み込み時、各サービスのステータスを取得
2. CPU使用率の初回データを取得
3. グラフに1点目をプロット

### 定期更新
1. 1分ごとにステータスとCPU使用率を取得
2. グラフにデータポイントを追加
3. ブラウザ側でデータを蓄積（最大60点 = 1時間分）

### エラー時
- 取得失敗時はステータスを「不明」として表示
- グラフは前回のデータを維持
