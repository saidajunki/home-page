# Meetings

議事録・ミーティングメモ

## 概要

ミーティング議事録、要約、決定事項を記録します。

## 注意事項

このディレクトリの議事録は仕様そのものではありません。
仕様のSource of Truthは `specs/` ディレクトリに格納されています。

ここに格納するのは、仕様書を作成するに至った経緯・やりとり・意思決定の背景などを証跡として残すためのものです。

## ディレクトリ構成

```
meetings/
└── YYYYMM/           # 年月ごとにディレクトリを分ける
    └── YYYYMMDD_*.md # 日付_ミーティング名.md
```

## 記載内容

- 参加者
- 議題
- 決定事項
- アクションアイテム
- 次回予定
